# Uso básico

## Abrir un PDF

Toca `Abrir PDF` y selecciona el archivo desde la app Archivos de iPadOS.

## Pasar de página

Usa los botones `Anterior` y `Siguiente` de la barra inferior. Entre ambos botones se muestra la página actual.

## Borrar zonas

Selecciona `Borrador`, ajusta el grosor con el control inferior y dibuja sobre el PDF.

Cada trazo se guarda como una sola acción de deshacer.

## Copiar y pegar una zona

1. Selecciona `Lazo`.
2. Rodea la zona que quieres copiar.
3. Toca `Copiar selección`.
4. La app cambia a `Pegar`.
5. Toca el punto del PDF donde quieres colocar la copia.

La primera versión del Lazo copia el rectángulo envolvente de la selección.

## Guardar cambios

Toca `Exportar PDF`. iPadOS abrirá el exportador del sistema para elegir nombre y ubicación.

Bleacher exporta un PDF nuevo; no sobrescribe automáticamente el archivo original.
